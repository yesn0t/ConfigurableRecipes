<font size="4">
<p>Adds recipes to the uncraftable items in the Minecraft. Those recipes can enabled/disabled trough config.</p>
<p>If you want to know how to use the config, check <a href="https://github.com/spacegoat3/ConfigurableRecipes/wiki/Tutorial">this page.</a></p>
<p>I am a new developer. So there is a high chance this mod is going to have a lot of <strong>bugs</strong> and <strong> errors</strong>. Please report to me if you have a problem, thanks.</p>

## FAQ
<p><span style="text-decoration: underline; font-size: 18px;"><strong>Will you port this mod to&nbsp;<em>Forge</em></strong>?</span></p>
<p><span style="font-size: 18px;">I am a new, inexperienced developer and in my opinion, Fabric is much more easier to code than Forge. Therefore a <strong>Forge Port is not planned for now</strong>. <span style="font-size: 14px;"><em>(If you want to port this mod to Forge, you can.)&nbsp;</em></span></span></p>
<p>&nbsp;</p>
<p><span style="text-decoration: underline; font-size: 18px;"><strong>Can I use this project in&nbsp;<em>X</em></strong>?</span></p>
<p><span style="font-size: 18px;"><strong>Of course</strong>. You can use this mod in every private or public project you are working on. Though, if your project is public, it would be nice if you credit me.</span></p>
<p>&nbsp;</p>
<p><span style="text-decoration: underline; font-size: 18px;"><strong>Will you port this mod to &nbsp;<&nbsp;<em><strong>1.17</strong></em>?</strong></span></p>
<p><span style="font-size: 18px;">Sorry but <strong>no</strong>. Minecraft changed its way of coding in the 1.17 version. I don't have the time and the resources to make a back-port.</span></p>
<p>&nbsp;</p>
